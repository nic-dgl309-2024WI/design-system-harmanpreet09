Grid and Flex classes

Class Description
.l-flex Displays an element as a flex container.
.l-flex-column Sets the flex direction to column.
.l-flex-row Sets the flex direction to row
.l-flex-wrap Applies flex wrapping to the container.
.l-justify-center Centers items on the main axis.
.l-align-center Centers items on the cross axis.
.l-grid Displays an element as a grid container.
.l-grid-columns-2 Defines 2 columns in the grid.
.l-grid-columns-3 Defines 3 columns in the grid.
.l-grid-rows-2 Defines 2 rows in the grid.
.l-grid-rows-3 Defines 3 rows in the grid.
.l-gap-4 Sets a 4px gap between grid/flex items.
.l-gap-8 Sets an 8px gap between grid/flex items.
