Spacing

Margin and Padding use

What property is for

  • m-for classes that set margin
  • p-for classes that set padding

Sides for margin and Padding

  • t- for classes that set margin-top or padding-top
  • b-for classes that set margin-bottom or padding-bottom
  • l-for classes that set margin-left or padding-left.
  • r-for classes that set margin-right or padding-right.
  • x-for classes that set margin-right and margin-left and padding right and padding left.
  • y-for classes that set margin-top and margin-bottom and padding top and padding bottom.

Sizes for margin and Padding

  • 0- for classes that delete the margin or padding by setting it to 0
  • 1- for classes that set the margin or padding 1rem.
  • 2-for classes that set the margin or padding 1.5rem.
  • 3-for classes that set the margin or padding 1.25rem.
  • 4-for classes that set the margin or padding 1.5rem.
  • 5-for classes that set the margin or padding 2rem.
